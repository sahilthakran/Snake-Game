# Snake Game 🎮 A classic Snake Game developed using Java and Swing. This project demonstrates Java GUI development, event handling, collision detection, and object-oriented programming concepts. --- 
#  Features - 
 Smooth snake movement - 
 Keyboard controls - 
 Apple generation system - 
 Collision detection - 
 Game Over screen - 
 Score growth system - 
 Java Swing based GUI ---

📌 Main Files Description
1. SnakeGame.java

This is the main entry point of the application.

Responsibilities:

Creates the main game window using JFrame
Adds the game board panel
Sets window properties
Launches the Snake Game
2. Board.java

This is the core game logic file.

Responsibilities:

Snake movement
Keyboard controls
Apple spawning
Collision detection
Rendering graphics
Game loop management
Game Over handling
🎮 Controls
Key	Action
↑	Move Up
↓	Move Down
←	Move Left
→	Move Right
▶️ How to Run
Clone Repository
git clone https://github.com/sahilthakran/Snake-Game.git
Open Project
cd Snake-Game
Compile Project
javac src/snakegame/*.java
Run Game
java -cp src snakegame.SnakeGame
📦 Run JAR File
java -jar SnakeGame.jar
